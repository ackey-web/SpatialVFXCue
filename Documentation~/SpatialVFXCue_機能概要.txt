================================================================================
  SpatialVFXCue v2.0 — 機能概要
  リアルタイム VFX 演出コントロールアセット for Spatial Creator Toolkit
================================================================================


■ SpatialVFXCue とは
────────────────────────────────────────────────────────────────────────────────
Spatial メタバース空間内でリアルタイムに VFX 演出を操作するための Unity アセット。
キーボード操作・マウス操作・BPM 同期を組み合わせ、VJ のようにエフェクトを
ライブでコントロールできます。

主な用途:
  - 音楽イベントでの VJ パフォーマンス（ビートに合わせた VFX 発動）
  - カウントダウンやチェーン演出によるステージ進行
  - 来場者参加型のインタラクティブ VFX（ボタン設置）
  - ホログラム VJ（Syphon/Spout 経由でプロジェクター出力と併用）


================================================================================
  コアシステム
================================================================================


■ VFXCueManager（核心スクリプト）
────────────────────────────────────────────────────────────────────────────────
キュー定義に基づき VFX Prefab をネットワークスポーン/消滅する中央管理。

機能:
  - キーボード入力による即時 VFX スポーン
  - 権限制御（スペースオーナー/管理者のみ操作）
  - 同時スポーン上限制御
  - ホールド機能（VFX の生成・保持・破棄）
  - クールダウン（連打防止）
  - 外部 API（名前/キーコードで発動）
  - エディタ環境では権限チェックを自動バイパス

キュー設定項目:
  トリガーキー / VFX Prefab / キュー名 / スポーン位置（Transform or 座標）/
  位置オフセット / 回転 / 自動消滅時間 / クールダウン / レイヤーインデックス


■ VFXCueEffect（VFX Prefab 用 NetworkBehaviour）
────────────────────────────────────────────────────────────────────────────────
VFX Prefab にアタッチするコンポーネント。

機能:
  - ネットワーク同期（全クライアントで ParticleSystem 再生）
  - 自動消滅（タイマー → 放出停止 → パーティクル消滅待ち → 破棄）
  - VJ パラメータ適用:
      SetIntensity()        … emission rate / start size / max particles
      SetSpeedMultiplier()  … simulation speed
      SetColorTint()        … start color（ベースカラーとの乗算）

  ※ ParticleSystem のみが対象。メッシュやアニメーターには影響しません。
     例: ドラゴン + スモーク → スモークの ParticleSystem だけが制御される


================================================================================
  BPM 同期システム
================================================================================


■ VFXCueBPMSync
────────────────────────────────────────────────────────────────────────────────
音楽のビートに合わせて VFX を自動発動する BPM 同期エンジン。

3つの発動モード:
  REPEAT  … キー ON/OFF トグルで毎拍自動リピート発動
  SINGLE  … キー押下 → 次の拍タイミングで1回だけ発動（クォンタイズ）
  HOLD    … キーを押している間だけ VFX 表示（離すと消える）
            BPM 同期中は拍の前半 ON / 後半 OFF で点滅

機能:
  - タップテンポ（T キー連打で BPM 自動検出、最大 8 タップ平均）
  - ビート開始/停止（B キー）
  - モード切替（M キー: REPEAT → SINGLE → HOLD → ループ）
  - ビートフラッシュインジケーター（4拍表示 + 点滅）
  - 外部 API: StartHold/StopHold, CycleMode, ToggleBeat, Tap


================================================================================
  VJ コントローラー
================================================================================


■ VFXCueVJController（パラメータ制御エンジン）
────────────────────────────────────────────────────────────────────────────────
VJ ミキサー的にパラメータを階層的に制御。

パラメータ階層:
  最終値 = キュー個別値 × レイヤー値 × マスター値

マスターパラメータ:
  - Master Intensity (0〜2)
  - Master Speed (0.1〜3)
  - Master Color Tint

レイヤーシステム（デフォルト 4 レイヤー）:
  All / BG（背景） / FG（前景） / Accent（アクセント）
  各レイヤー: intensity / speed / mute / solo / color tint

  ミュート … そのレイヤーの VFX を非表示
  ソロ     … そのレイヤーだけ表示、他は全ミュート

キュー個別パラメータ:
  各キューに個別の intensity / speed / color tint を設定可能

プリセット機能（最大 8 スロット）:
  - マスター値 + 全レイヤー状態 + 全キュー個別値を一括保存/リコール
  - P キーで保存、Q/W/E/R でリコール

ビートカラーサイクル:
  - 有効化すると拍ごとにカラーパレットを自動巡回
  - V キーで ON/OFF

キーボード操作:
  N        … 選択モード ON/OFF（数字キーでキュー選択）
  ↑/↓      … 選択キューの intensity 増減（長押し対応）
  ←/→      … 選択キューの speed 増減（長押し対応）
  C        … カラーパレット次へ
  L        … レイヤー選択（次へ）
  K        … ミュートトグル
  J        … ソロトグル
  Tab      … VJ パネル表示/非表示
  H        … ヘルプ表示


■ VFXCueVJPanel（VJ コントローラー UI）
────────────────────────────────────────────────────────────────────────────────
画面下部に横長で表示される VJ コントローラーバー。
マウスクリック＆ドラッグで全操作可能。

4セクション構成:

  ┌──────────┬────────────┬──────────┬─────────────────────────────┐
  │ CUE PADS │LAYER MIXER │  MASTER  │         TRANSPORT           │
  └──────────┴────────────┴──────────┴─────────────────────────────┘

  CUE PADS（左端）
    - 2×3 グリッドのパッドボタン
    - 左クリック: VFX 発動（BPMSync モード対応）
    - 右クリック: キュー選択
    - HOLDモード: マウス押下中のみ VFX 表示、離すと消える
    - アクティブ時: レイヤーカラーで発光

  LAYER MIXER（中央左）
    - 4 本の縦型フェーダー（各レイヤーの intensity）
    - マウスドラッグで値変更
    - M/S ボタン（ミュート/ソロ）

  MASTER（中央右）
    - Intensity / Speed の縦型フェーダー 2 本
    - マウスドラッグで値変更
    - 数値表示

  TRANSPORT（右端）
    - プリセットボタン 1〜4（クリックでリコール）+ SAVE
    - BPM 表示 + TAP + ON/OFF
    - モード切替ボタン（REPEAT/SINGLE/HOLD）
    - ビートインジケーター（4拍）
    - カラーパレット（クリックで色選択）

  UI技術: GUIUtility.hotControl による確実なマウスイベント捕捉


================================================================================
  演出補助スクリプト
================================================================================


■ VFXCueChain（チェーン演出）
────────────────────────────────────────────────────────────────────────────────
複数の VFX キューを時間差で自動連鎖発動。

  - Inspector でステップ一覧を定義（キュー名 + 遅延秒数）
  - ループ再生対応
  - キー 7 で開始/停止
  - 外部 API: StartChain() / StopChain()

用途: セットリスト的な演出シーケンス


■ VFXCueRandom（ランダム VFX ルーレット）
────────────────────────────────────────────────────────────────────────────────
キュープールからランダムに 1 つの VFX を発動。

  - 連続同一 VFX 回避オプション
  - クールダウン制御
  - キー 8 で発動
  - 外部 API: TriggerRandom()

用途: 来場者向けガチャ演出


■ VFXCueCountdown（カウントダウン演出）
────────────────────────────────────────────────────────────────────────────────
3... 2... 1... GO! のカウントダウン後に全 VFX を一斉発動。

  - 空間内に 3D テキスト（TextMesh）でカウント表示
  - ポップイン＋フェードアウトアニメーション
  - テキストがローカルアバターの方を向く（ビルボード）
  - クライマックス VFX の時差発動（stagger 設定）
  - キー 9 で発動
  - 外部 API: StartCountdown() / CancelCountdown()


■ VFXCueTriggerButton（インタラクティブボタン）
────────────────────────────────────────────────────────────────────────────────
空間内に設置するクリック可能な VFX 発動ボタン。

  - SpatialInteractable と組み合わせて使用
  - 来場者が F キーまたはクリックで VFX を発動
  - バウンスアニメーションフィードバック
  - ボタン個別のクールダウン

用途: 来場者参加型の演出


■ VFXCueLog（ランタイムログ）
────────────────────────────────────────────────────────────────────────────────
画面左下に VFX 発動ログをフェード表示するデバッグ用コンポーネント。
VFXCueManager と同じ GameObject にアタッチするだけで有効。

  同梱アセット
================================================================================

  Prefabs/（サンプル VFX 6 種）
    VFX_Fireworks       … 打ち上げ花火（開花パーティクル）
    VFX_Confetti        … 多色紙吹雪
    VFX_MeteorShower    … 流星群（上空から降り注ぐ）
    VFX_SpotlightBeam   … 光柱ビーム
    VFX_SmokeBurst      … スモーク爆発
    VFX_SparkleRain     … キラキラ粒子の雨

  Materials/（URP Particles シェーダー 6 種）
  Textures/（Circle / Star / Streak / Smoke の 4 テクスチャ）
  Scenes/SpatialVFXCue_Demo.unity（デモシーン）

  Editor/
    VFXCueSetupWizard     … メニューからサンプルアセット自動生成
    VFXCueManagerEditor   … カスタム Inspector UI
    VFXCueImportHelper    … インポート時の自動セットアップ

  - Universal Render Pipeline（URP）
